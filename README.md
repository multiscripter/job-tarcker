**Проект:** Job tracker

Выводит задачи в древовидном 
представлении:
```
годы
    месяцы
        день 1          время (чч:мм)
        день 2          время (чч:мм)
        . . .
        день N          время (чч:мм)
    (время мес.) макс чч (%)  тек. чч
(время годы)     макс чч (%)  тек. чч
```
**Окружение:**

Каркас: **Django 3.2**

База данных: **SQLite**

**Запуск из корня проекта:**
```
python manage.py runserver
```
**Адрес проекта:**
```
http://127.0.0.1:8000/
